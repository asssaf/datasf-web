# Tech Stack

## Programming Language
- **Python**: A powerful and flexible language that's ideal for rapid application development and has extensive support for API interactions.

## Frameworks and Libraries
- **CLI Framework**: **Click** (Python) - A user-friendly and feature-rich library for building robust command-line interfaces.
- **HTTP Client**: **Requests** (Python) - The industry-standard library for making clean, reliable HTTP requests to REST APIs.

## Build and Distribution
- **Static Binary Compilation**: The application will be compiled into a single, standalone binary (e.g., using PyInstaller or Nuitka) to ensure easy distribution and use without requiring a Python interpreter on the target system.
